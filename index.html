⚙️ Konfigurasi JSONBin

Karena akun sekolah dibatasi, kita gunakan JSONBin.io sebagai perantara gratis. Apps Script di sekolah mengirim data ke JSONBin tiap 5 menit, website ini membacanya.

Google Form

Website Ini ✅
1
Daftar gratis di jsonbin.io → Login → klik API Keys → Create Access Key → copy key-nya
2
Klik Bins → + New Bin → isi dengan {"rows":[]} → klik Create → copy BIN ID dari URL
3
Di Apps Script sekolah: isi JSONBIN_API_KEY dan JSONBIN_BIN_ID → jalankan syncToJsonBin → lalu jalankan setupTrigger
4
Isi BIN ID di bawah → klik Simpan. API Key tidak perlu diisi di website (hanya dipakai di Apps Script)
BIN ID tidak boleh kosong.
